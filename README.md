# M6 Sprint Grupal

Sistema de gestión de resultador Formula 1.

## Requerimientos 
* Mysql 

## How to

Antes de iniciar crear un archivo `config-db.json`, con la información de conexión a la base de datos:

```json
{
  "DB_HOST": "localhost",
  "DB_PORT": 3308,
  "DB_USER": "root",
  "DB_PASS": "root",
  "DB_NAME": "m6_sprint"
}
```

Para iniciar el servidor utilizar 

```bash
npm run dev
```
